Apache HTTP server

This page referenced the guide from DigitalOcean and Ubuntu.

Install Apache by running:
sudo apt update
sudo apt install apache2
Install ufw
View ufw application profiles: 
sudo ufw app list
 The output looks like this: 
#output
Available Applications:
	Apache
	Apache Full
	Apache Secure
 The application profiles are located at /etc/ufw/applications.d.
The profiles contain the display name of the profile, the title of the app, a description, and the port that the app uses.
Here is the list of port(s) associated with the Apache profiles: 
#list
Apache          80/tcp
Apache Full     80, 443/tcp
Apache Secure   443/tcp
 We open the http and https ports using the command: 
sudo ufw allow 'Apache Full'
 Enable ufw and check the running status: 
sudo ufw enable
sudo ufw status
 
#output
Status: active
 
To			Action		From
--			------		----
Apache Full		ALLOW		Anywhere
Apache Full		ALLOW		Anywhere (v6)
By default, Apache would start running right after you install it. 
sudo systemctl status apache2
 Output should indicate the system is active (running).
